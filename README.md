v22 — «Старт и возвращение»
--------------------------------

БЛОК A — СТАРТ (активация, «первая победа за 10 минут»)

A1. «Первый ролик за 10 минут» — один проводящий поток для новичка без канала: ниша → 3 идеи → 1 готовый сценарий → упаковка → чек-лист публикации. Цель — человек уходит с сайта с готовым первым роликом уже в первый визит. Подхватывает уже готовые v21-модули.
A2. «7-дневный челлендж запуска» — пошаговый план «от 0 до первых 3 роликов за неделю» с заданием на каждый день (день 1 — ниша, день 2 — 10 идей, день 3 — сценарий…). Снимает страх «с чего начать» — главный барьер новичка.

БЛОК B — ВОЗВРАЩЕНИЕ (ретеншн, «причина вернуться»)

B1. Стрик + недельная цель — «выкладывай N роликов/нед», счётчик стрика, прогресс-бар недели, мягкий напомин-нажим при заходе. Люди возвращаются, чтобы не порвать стрик. (localStorage)
B2. Трекер роста канала — мини-дашборд «ты вырос на X% за неделю»: подписчики/просмотры по неделям, график, мини-выводы. Ручной ввод или из подключённого канала (OAuth/Analytics уже встроен). Повод заходить раз в неделю «свериться».
B3. Продакшн-доска — апгрейд «Моих съёмок» в канбан идея → сценарий → снято → опубликовано с прогрессом. Человек видит, что движется → возвращается доделать.
B4. Ачивки + уровень продюсера — бейджи (первый ролик, стрик 7 дней, 10 опубликовано), уровни. Лёгкая геймификация держит новичка в игре.
